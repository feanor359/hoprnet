Struct NoSumSMA

source 
pub struct NoSumSMA<T, D = T> { /* private fields */ }
Expand description

Basic implementation of Simple Moving Average (SMA).

The maximum window size is bound by 2^32 - 1. Useful mainly for floating-point types, as it does not accumulate floating point error with each sample. Requires O(N) of memory and O(N) for average computation, N being window size. The divisor argument D is used only for such types T that do not implement From<u32> (such as Duration,…).
