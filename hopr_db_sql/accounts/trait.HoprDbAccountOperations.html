Expand description

Defines DB API for accessing HOPR accounts and corresponding on-chain announcements.

Accounts store the Chain and Packet key information, so as the routable network information, if the account has been announced as well.

If an account matching the given key (chain or off-chain key) does not exist, an error is returned. If such multiaddr has been already announced for the given account key, only the at_block will be updated on that announcement.
