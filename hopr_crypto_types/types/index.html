Implements basic cryptography related types based on primitives, such as Hash, PublicKey and Signature.

  • HalfKeyChallenge

    Represents an ECDSA signature based on the secp256k1 curve with recoverable public key. This signature encodes the 2-bit recovery information into the uppermost bits of MSB of the S value, which are never used by this ECDSA instantiation over secp256k1. The instance holds the byte array consisting of R and S values with the recovery bit alredy embedded in S.
